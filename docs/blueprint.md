# **App Name**: WanderLink

## Core Features:

- User Profiles: Create and manage user profiles with travel preferences and experiences.
- Group Formation: Allow users to create and join travel groups based on shared interests and destinations.
- Destination Discovery: Explore various travel destinations with curated information and user reviews.
- Travel Planning: Facilitate travel planning with date selection, budget estimation, and itinerary suggestions, leveraging a tool to provide tailored information based on the groups' past discussions..
- Reminders: Set and receive reminders for upcoming trips and important travel dates.
- Budgeting Tool: Help estimate and track travel expenses with budget planning features.
- Chat Section: Enable real-time communication within travel groups for easy coordination.

## Style Guidelines:

- Primary color: Sunset Orange (#FF8C42) to evoke a sense of adventure and warmth.
- Background color: Light Sand (#F5F5DC), a very desaturated hue similar to orange to create a welcoming and neutral canvas.
- Accent color: Teal Green (#008080), to contrast and highlight interactive elements, evocative of global waters and forests.
- Font pairing: 'Belleza' (sans-serif) for headlines, and 'Alegreya' (serif) for body text, for a fashionable yet readable look.
- Use custom line icons to represent destinations, activities, and travel tools.
- Employ a clean, card-based layout to showcase destinations and user profiles.
- Incorporate subtle animations for loading states and interactive feedback.
